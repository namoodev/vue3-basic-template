# Hello World!

> 이 프로젝트는 vue에 입문하는 입문자들을 위한 샘플 및 가이드 프로젝트입니다. 하나의 프로젝트에 모든 가이드를 작성하는 것 보다, 개별 프로젝트로 생성하는 것이 보다 더 가독성이 있을 것 같아 개별 프로젝트로 진행합니다.

> 서로 영향을 주고받는 라이브러리 혹은 라이브러리 응용을 통한 다른 코딩 스타일 역시 가이드에 녹여낼 예정입니다.

## 프로젝트 요약 설명

```plaintext
vue3-basic-template/
├── README.md               # 전체 학습 로드맵 및 요약
├── 01_basic_init/          # `vue create folderName` 으로 생성한 초기 프로젝트
├── 02_basic_vue-router/    # `vue-router` 초기 세팅 및 간단 샘플 프로젝트
└── 03_...                  # 이후 추가될 프로젝트들 (03, 04, 05, ...)
```
